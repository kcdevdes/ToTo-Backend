# ToTo application backend

> https://foremost-blender-ac2.notion.site/TOTO-1784947e81224eb78d7bdf468bc7f51c

* Database : MongoDB

# How to use this?
```bash
# Clone this repository
$ git clone https://github.com/kcdevdes/ToTo-Backend
# Install node dependencies
$ npm install
# Run server
$ npm run start
```
